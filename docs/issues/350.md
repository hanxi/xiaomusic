# 播放本地歌曲无法切歌

播放本地歌曲时，通过语音控制小爱音箱切歌（测试指令：小爱同学，切歌；小爱同学，播放下一首；），每次都是重新播放上一首歌曲；版本为docker最新版本[0.3.70]
