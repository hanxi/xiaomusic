# 同步网易云歌单

推广一波 [netease-playlist](https://github.com/qiujie8092916/netease-playlist)  
通过网易云音乐歌单 ID，下载歌曲或[生成歌单配置](/issues/269.html)  
本程序为独立模块，可 docker 部署，可作为不用[插件](/issues/105.html)的另外实现方式
