# 使用cookie登陆

1. 在电脑上使用 Chrome 浏览器登录小米账号官网 👉 <https://account.xiaomi.com/>
2. 按 `F12` 打开网络工具，点击 `Network` 标签页。
3. 点击左上角小圆点使其变红色。
4. 按 `F5` 刷新页面。
5. 在过滤框里输入 `account?` 过滤出下面的连接，点击连接查看 `cookie` ，复制完整的 `cookie` 值。需要确保复制的  `cookie` 中有 `userId` 和 `passToken` 两个值。
