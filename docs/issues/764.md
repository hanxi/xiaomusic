# 开发了酷狗音乐歌单导出工具，可快速导入 xiaomusic

你好！

我开发了一个酷狗音乐歌单导出工具，可以将酷狗音乐的歌单导出为 xiaomusic 兼容的 JSON 格式。

项目地址： https://github.com/Steven-Qiang/kugoumusic-playlist-exporter

主要功能：

支持手机验证码、二维码扫码登录

一键导出个人收藏歌单为 xiaomusic 格式

自动获取歌曲播放链接

技术实现：
基于 [KuGouMusicApi](https://github.com/MakcRe/KuGouMusicApi) 开发，提供了完整的 Web 界面和可执行文件。

导出的 JSON 文件可直接在 xiaomusic 设置页面导入使用。

希望能在 README 中提及这个工具，方便用户使用酷狗音乐歌单。

感谢！
