# 局域网 ip 问题，排查了两个小时。希望能对大家有所帮助

家里使用了两个路由器 

- A 路由器 192.168.1.x
- B 路由器 192.168.31.x

-  在电脑上 docker 启动，连接`B路由器的网络` 要将 `XIAOMUSIC_HOSTNAME(IP或域名):`这一项设置为`B路由器的网络` 也就是`192.168.1.x`
-  A路由器同理
